# MPG Document Source Layout

This directory contains all maintained document input files.

## Conventions

- Keep chapter filenames stable (for example `m-started.tex.in`).
- Place chapter files under `chapters/<domain>/`.
- Keep shared TeX includes under `static/`.
- Keep bibliography inputs under `bib/`.
- Keep non-TeX source assets under `assets/`.
- Keep free-form notes under `notes/`.

## Chapter Domains

- `chapters/core/`
- `chapters/modeling/`
- `chapters/case-studies/`
- `chapters/programming/`
- `chapters/search/`
- `chapters/appendix/`
- `chapters/meta/`
